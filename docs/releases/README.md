# ZoneWise Release Notes

Auto-generated by upstream sync pipeline.
Each file maps a Craft Agents OSS release to a ZoneWise V2 release.

## Naming Convention
`ZONEWISE_V{version}_RELEASE_{date}.md`

## Pipeline
1. Craft Agents releases new version
2. sync-upstream.yml detects within 6 hours
3. Code synced, feature parity verified
4. Release doc auto-generated
5. Web + Desktop deployed
6. GitHub Release created
